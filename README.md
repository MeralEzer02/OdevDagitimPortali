# 📚 Ödev Yönetim Sistemi (LMS)

Öğretmen ve öğrenciler için geliştirilmiş, API + MVC tabanlı modern eğitim yönetim sistemidir.

---

## 🎯 Amaç
Ödev oluşturma, teslim etme ve değerlendirme süreçlerini dijitalleştirmek.

---

## 🛠️ Teknolojiler
- ASP.NET Core Web API
- ASP.NET Core MVC
- MSSQL
- Entity Framework Core
- ASP.NET Identity
- JWT Authentication
- jQuery AJAX
- Bootstrap 5

---

## 🔐 Güvenlik
- JWT tabanlı authentication
- Role-based authorization (Admin / Teacher / Student)
- Veri sahipliği kontrolü
- Global exception handling

---

## ⚙️ Özellikler
- Ders ve ödev yönetimi
- Dosya yükleme sistemi
- Ödev teslim ve değerlendirme
- Rol bazlı panel sistemi
- API + MVC entegrasyonu

---

## 🧱 Mimari
- Clean Architecture yaklaşımı
- Repository + Service Layer
- DTO tabanlı veri akışı
